Phaser.js와 Babylon.js를 활용한 2D/3D 슈팅 액션 게임 【젠더리스 에러】

바로가기

프로젝트 개요

【젠더리스 에러】는 2025년 2월에 개발한 HTML5 기반의 웹 게임입니다.

장르는 '성별'이라는 주제를 중점으로 한 SF, 테크노 스릴러 장르의 줄거리를 담고 있으며, 게임의 장르는 비주얼 노벨, 슈팅 액션 게임입니다.

JavaScript 기반의 게임 엔진인 Phaser.js와 JavaScript 기반의 3D 그래픽 작업을 지원해주는 Babylon.js를 활용했습니다.

위 자료 사진과 같이 모바일 접속 환경도 지원합니다.
